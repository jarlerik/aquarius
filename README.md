
# Rails + React web application for Aquarius sporting club

# System requirements

* Ruby 2+

* Rails 5+

* Bundler

* PostgreSQL

* Node (+npm)

# Up and running

* bundle install

* Creating an optimized production build

  1 cd client

  2 npm run build

  3 cd ..

  4 cp -a client/build/. public/

* Starting development server

  - rake start
